## 介绍

此markdown编辑器是一个基于Android平台的应用，提供编辑和Markdown预览功能，以及支持Markdown文件操作。

## 特点

- 轻量化
- 易上手
- 界面简洁
- 可二次开发
  
## 使用场景

- 技术文档编写
- 日常记录
- 学习和借鉴软件开发
